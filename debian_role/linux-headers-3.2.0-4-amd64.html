Header files for Linux 3.2.0-4-amd64

This package provides the architecture-specific kernel header files for Linux kernel 3.2.0-4-amd64, generally used for building out-of-tree kernel modules. These files are going to be installed into /usr/src/linux-headers-3.2.0-4-amd64, and can be used for building modules that load into the kernel provided by the linux-image-3.2.0-4-amd64 package.
